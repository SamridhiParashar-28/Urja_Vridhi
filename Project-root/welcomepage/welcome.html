// about the project

What is WattWise?

WattWise is a web-based energy monitoring platform developed as part of this project.

The system supports native CSV parsing, runs a hybrid LSTM + XGBoost forecasting model, and provides secure **End-to-End Encryption (E2EE)** for sensitive dataset management via AES-256-GCM. An intelligent **Gemini-powered AI Assistant** is integrated to provide context-aware energy analysis and actionable optimization tips.

Built on a modern **Node.js + Express** backend with a high-performance **SQLite** database, the project features a custom **Blocks Manager** for dynamic environment configuration and secure OTP-based sharing rules.

CSV Data Upload Upload meter readings per block, room and appliance. Supports full dataset format.
Live Usage Monitoring Zone-level real-time consumption view across all monitored blocks, refreshing automatically.
Short-Term Forecasting LSTM + XGBoost hybrid model predicts next-day kWh per block with 94.2% accuracy.
AI Energy Assistant Gemini-powered multi-turn chat for context-aware energy insights and saving tips.
E2EE Dataset Security Sensitive data is encrypted client-side using AES-256-GCM before storage.
Dynamic Blocks Manager Create and share custom environments with secure OTP-based passkeys.
Anomaly Detection Automatically flags unusual spikes and baseline deviations with severity ratings.
Billing & Cost Simulation Detailed cost estimates at configurable tariff rates for all blocks.
